Error al cargar la imagen

La imagen desde Google Drive no se pudo cargar. Esto sucede porque Google Drive bloquea el acceso directo desde páginas web.

Solución: Por favor, sube la imagen a un servicio como imgBB.com y reemplaza el enlace en el código.

Para usar tu imagen en GitHub Pages:

1. Sube tu imagen a imgBB.com (gratis)

2. Copia el enlace directo que te den

3. Reemplaza esta línea en el código (línea 348):

src="https://drive.google.com/uc?export=view&id=1fZNdMvKzHbC3JLP_e8ljIjFHLBZOVgjp"

4. Sube el archivo HTML actualizado a GitHub
